## Development - Build

### Requirement
* `git` installed
* `node` installed
  * `npm` installed


### Build locally
Clone the repository using
```bash
git clone git@github.com:MattixNow/mattaio-website.git
```

Install the dependencies
```markdown
npm install
```

Start the Nuxt.js devloppement server
```bash
npm run dev
```
